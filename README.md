# desktop-focus-candle
A minimalist 3D-styled desktop candle timer built with Python and Tkinter for focus and productivity.


Features: 3D burning animation, melting effect, smoke particles, and auto-closing timer.

How to Run: python main.py komutuyla çalıştığını belirt.

Visuals (Çok Önemli): Mumun bir ekran görüntüsünü veya bir GIF'ini mutlaka README'ye ekle. İnsanlar koddan önce nasıl göründüğüne bakacaktır.
